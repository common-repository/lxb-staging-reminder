== Description ==

Give the user a visual reminder that he or she is on a wp-engine staging environment.

== Installation ==

1. Upload `lx-staging-reminder` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
